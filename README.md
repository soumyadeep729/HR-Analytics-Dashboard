# HR-Analytics-Dashboard
Designed and developed an interactive HR analytics dashboard in Power BI to analyze employee
attrition across age groups, education levels, and salary slabs.Implemented KPIs, slicers, and dynamic visualizations to identify key attrition drivers and support
data-driven workforce retention strategies.
